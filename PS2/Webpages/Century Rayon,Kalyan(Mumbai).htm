PROBLEM BANK DETAILS

Problem No. 1 
Problem Title AT PRESENT BITS STUDENTS IS WORKING FOLLOWING PROJECT: STUDY AND ENHANCEMENT OF THERMAL EFFICIENCY OF DRYERS AND EVAPORATORS IN RAYON PLANT 
Problem
Description		 THE TASK AT HAND IS TO ANALYZE THE CURRENT WORKING CONDITIONS OF THE DRYERS AND EVAPORATORS IN RAYON PLANT AND CALCULATE THE EFFICIENCY WITH WHICH THEY ARE CURRENTLY WORKING. FOLLOWING THIS, IS TO ENHANCE THE THERMAL EFFICIENCIES, NOT ONLY TO AMELIORATE THE WORKING CONDITIONS BUT ALSO TO REDUCE STEAM CONSUMPTION IN THE TARGETED AREAS. 
Skill Set ANALYTICAL MIND, APTITUDE TO RESEARCH, HARD WORKING 

Problem No. 2 
Problem Title AT PRESENT BITS STUDENTS IS WORKING ON FOLLOWING PROJECTS: STATISTICAL ANALYSIS OF VISCOSITY AND RIPENING INDEX OF VISCOSE FOR THE YEAR 2011, TYRE CORD DEPARTMENT. 
Problem
Description		 THE VISCOSITY AND RIPENING INDEX OF VISCOSE PLAY A MAJOR ROLE IN DETERMINING THE PHYSICAL PROPERTIES OF THE TYRE CORD PRODUCED. THIS PROJECT FOCUSES ON SUGGESTING METHODS FOR A CLOSER CONTROL OF THESE PROPERTIES. THE SEVEN BASIC TOOLS OF QUALITY CONTROL (ISHIKAWA DIAGRAM, CHECK SHEET, CONTROL CHART, HISTOGRAM, PARETO CHART, STRATIFICATION, SCATTER DIAGRAM AND SIPOC ANALYSIS) HAVE BEEN USED TO STUDY THE VARIATIONS IN VISCOSITY AND RIPENING INDEX. THE INFERENCES DRAWN HERE ARE ANALYZED IN DETAIL. 
Skill Set ANALYTICAL MIND, APTITUDE TO RESEARCH, HARDWORKING 

Problem No. 3 
Problem Title AT PRESENT BITS IS WORKING FOLLOWING PROJECT: ZINC RECOVERY FROM CALCINATION SYSTEM OF TYRECORD & REDUCTION OF ZN LEVELS IN THE COMBINED EFFLUENT STREAMS OF THE CSY AND TYRECORD PLANTS. 
Problem
Description		 THE ZN LOSSES ON A DAILY BASIS WERE CALCULATED AND ANALYZED. AN ALTERNATIVE TO THE CURRENT SYSTEM OF ZN RECOVERY HAS BEEN PROPOSED. THIS INVOLVES INSTALLING A REACTOR WHERE ZINC IS ALLOWED TO REACT WITH SODA ASH. SUBSEQUENTLY, THE REACTED STREAM WILL BE FED TO A CONTINUOUS CLARIFIER WHERE ZINC CAN BE SEDIMENTED AS ZNCO3. THE SEDIMENT CAN BE DECANTED AT REGULAR INTERVALS AND REACTED WITH SULPHURIC ACID TO REGENERATE ZINC SULPHATE.  
Skill Set ANALYTICAL MIND, APTITUDE TO RESEARCH, HARD WORKING
